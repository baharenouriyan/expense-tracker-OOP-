# expense-tracker-OOP-

: Expense Tracker (Python)

A simple and practical Expense Tracker built with Python that allows users to record income and expenses, store data permanently using JSON, and calculate their account balance automatically.

This project is perfect for beginners who want to practice:

- File Handling
- JSON data storage
- Functions
- Lists & Dictionaries
- User Input
- Program Logic

---

:rocket: Features

:white_check_mark: Add new transactions (income or expense)
:white_check_mark: Automatically save data to a JSON file
:white_check_mark: View all transactions
:white_check_mark: Calculate and display current balance
:white_check_mark: Prevent program crash if the file does not exist
:white_check_mark: Clean and beginner-friendly code
